## 2024-05-13 - Focus Styles and Keyboard Events for Custom Elements
**Learning:** In ArcGIS applications, relying solely on `tabindex` for interactive `<div class="ecard">` elements isn't enough; they require an explicit `keydown` listener checking for `Enter` and `Space` to function like native `<button>` tags. Furthermore, `outline-offset` is critical: positive for free-floating controls (`.ctrl`, `#popupClose`) and negative (`-2px`) for bounded grid/list elements (`.ecard`) to prevent clipping within `overflow: hidden` containers. Brand color Volt (`#ccff00`) performs exceptionally well for high-contrast focus rings on dark themes.
**Action:** Always bind a `keydown` handler when using `tabindex="0"` on non-native interactive elements, and apply context-aware `outline-offset` values to ensure the focus state remains visible.
